Sisaweb Data

Dados extraídos via API do Sisaweb

Sobre os dados

Os dados apresentados foram obtidos diretamente da API do Sisaweb, gentilmente disponibilizados através do Moodle da disciplina. Agradecemos profundamente aos dedicados trabalhadores e equipes que coletaram e organizaram essas informações, essenciais para o enfrentamento de desafios de saúde pública. Nas imagens exibidas, é possível observar mapas de calor que evidenciam as notificações, áreas de maior foco e os locais onde esforços foram concentrados no combate ao mosquito Aedes aegypti. Esses dados são fundamentais para compreender a dinâmica da transmissão de doenças como dengue, zika e chikungunya, além de embasar estratégias mais eficazes de controle.
